Skills

已收录的技能

每个 Skill 是一套完整的交互流程定义，可独立嵌入 AI IDE 配置中使用。

⛓

Prompt Chaining

将复杂编程任务拆解为有序阶段，逐步精炼，获得高质量结果。

分析 → 生成 → 选择 → 补充 → 综合
  • 先理解上下文，再动手写代码
  • 强制产出多方案并对比选优
  • 根据任务规模自动降档/升档
  • 内置 5 种场景模板 + IDE 集成指南
✎

DrawIt

交互式概念可视化，通过对话将抽象想法变成直观的图形。

理解意图 → 选择方案 → 绘制迭代
  • 支持 Mermaid / SVG / HTML / Excalidraw
  • 15 种 Mermaid 图表类型速查
  • 10 种场景绘图模板开箱即用
  • 三套配色方案（专业/简洁/暗色）

Skill 1

Prompt Chaining — 链式提示

五阶段流程：分析现状，生成方案，择优选择，补充细节，综合执行。遇到问题时可回溯到任意阶段。

    B["2. 生成\nGenerate"]
    C["3. 选择\nSelect"]
    D["4. 补充\nSupplement"]
    E["5. 综合\nSynthesize"]

    A --> B --> C --> D --> E

根据规模自动适配

精简模式

改动 < 3 个文件时，合并分析+生成+选择，快速定位方案。

标准模式

3-10 个文件，完整五阶段流程，每阶段简洁但不省略。

深度模式

> 10 个文件，每阶段详细展开，增加架构评审，考虑分批实施。

Principles

核心原则

1
逐步精炼优于一步到位
分阶段处理让每一步都更可控
2
显式推理优于隐式假设
把思考过程外化，减少遗漏
3
多方案对比优于单一路径
比较才能发现各方案的优劣
4
上下文感知优于模板套用
每个项目都有自己的约定，需要尊重
5
灵活适配优于僵化执行
流程为人服务，不是人为流程服务
